*** Mumble

  #+begin_src shell
    mkdir ~/.local/containers/mumble/data-volume

    podman volume create \
           --opt type=none \
           --opt device=$HOME/.local/containers/mumble/data-volume/ \
           --opt o=bind \
           mumble-data

    podman run -d \
           --name mumble-server \
           --pod=botcodes \
           -v mumble-data:/data:Z \
           -v $HOME/.local/containers/letsencrypt:/etc/letsencrypt:ro \
           -e "MUMBLE_CONFIG_PORT=8007" \
           docker.io/mumblevoip/mumble-server

    #+end_src

** Jitsi
Following instructions here for [[https://jitsi.github.io/handbook/docs/devops-guide/devops-guide-docker/#internal-authentication][Internal authentication]]

Run the following commands after systemd has successfully created the containers for jitsi-meet.
  #+begin_src shell
    podman exec -it jitsi-prosody /bin/bash
  #+end_src

Once in the container, run the following command to create a user
  #+begin_src shell
  podman exec -it jitsi-prosody prosodyctl --config /config/prosody.cfg.lua adduser bot@meet.jitsi
  #+end_src
